# Install:

```git clone```  
```npm i```

# Use with OSpanel  

### Add domain  

| Domain        | Path            |
| ------------- |:---------------:|
| domain-name   | domain-name/src |

### Edit hostname on build/webpack.base.conf.js

``` const HOSTNAME = 'domain-name' ```

### Dev mode  
```npm run dev```
  
### Prod mode  
```npm run build```
